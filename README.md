# Gitlab-CI-CD

2. High-Level Principles

Collaboration: Breaking down silos; Developers and Operators share responsibility for the product.

Automation: Removing manual "toil" to increase speed and consistency.

Continuous Feedback: Implementing short feedback loops to catch errors immediately (Fail Fast).

3. CI/CD: The "Engine" of DevOps

Continuous Integration (CI): The practice of automating the Build and Test stages.

Principle: "Merge often, test everywhere."

Continuous Deployment (CD): The practice of automating the Release and Deploy stages.

Principle: "Release reliable code to customers on demand."
